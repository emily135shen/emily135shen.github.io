What the DOM is and how you might interact with it.

The DOM is what allows our browsers to display content, style and structure. In other words, it’s the application programming interface to HTML and XML documents. The DOM is made up of nodes. Nodes are elements, text content, comments, attributes etc. HTML elements that are accessed is taken from the base of the node which displays all nodes. One might open the DOM to edit the styling or html.

"The W3C Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document."

var myData = [0];
